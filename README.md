# 📊 Sales Performance Dashboard (Excel)

An interactive, high-performance sales dashboard built entirely in Excel. This project provides a comprehensive overview of sales executive performance, regional trends, and target achievement metrics.

---

## 🚀 Overview

This dashboard is designed to provide actionable insights into sales data. It allows users to filter performance by city, toggle between different dashboard views, and visualize key performance indicators (KPIs) through dynamic charts and tables.

## 📈 Key Features & Metrics

### 1. Performance Tracking
- **Total Sales by Executive**: Real-time tracking of sales volume generated by each team member.
- **Target Hit %**: A percentage-based view of how close executives are to their set goals.
- **Variance Analysis**: Tracking how far executives are "Away from Target" to identify areas for improvement.

### 2. Interactive Insights
- **Regional Filtering**: Slicers for multiple cities (Chennai, Delhi, Mumbai, etc.) allow for localized analysis.
- **View Toggles**: Custom checkboxes to switch between different dashboard perspectives (Dashboard 1-4).
- **Dynamic Visuals**:
    - **Horizontal Bar Charts** for ranking executives.
    - **Donut Charts** for market share/contribution analysis.
    - **Line Charts** for trend and variance tracking.

### 3. Advanced Excel Techniques
- **Pivot Tables & Charts**: Backend data processing for fast, responsive updates.
- **Slicers & Timelines**: User-friendly interactive elements for data navigation.
- **Custom Formatting**: High-contrast dark theme for a professional "modern app" feel.

## 🛠️ How to Use
1. Download the `EXCEL PROJECT 1.xlsm` file.
2. Ensure macros are enabled (if applicable) for the best experience.
3. Use the **City Slicers** at the top to filter data.
4. Use the **Dashboard Checkboxes** to explore different data views.
